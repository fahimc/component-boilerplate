Small Footprint

Component.js is 1 - 6kb in file size on it's own. Since this framework is modular if you add the router and have es5 support then the file size is only 9kb!

Natural

This framework is designed similar to Vuejs so if you've used Vuejs you will be familar with this.
It focuses on SOC (separation of concern), HTML templates are in HTML, JS is within a JS file and Sass is used for css.

Modular

The concept of this framwork is to keep this minimal and it plays well with others. You can bolt on other libraries to add more features
