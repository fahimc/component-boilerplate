Small Footprint

Component.js is only 1 - 6kb in file size. This framework is modular allowing to add many other plugins on top of it like a router. This together with es5 support, the file size is only 9kb!

Natural

This framework is designed similar to Vuejs so if you've used Vuejs you will be familiar with this.
It focuses on SOC (separation of concern), HTML templates are in HTML, JS is within a JS file and Sass is used for css.

Modular

The concept of this framework is to keep things minimal and needs plays well with others. You can bolt on other libraries to add more features

Fast

Using the create task, you can instantly generate a component. All the files are created with the correct structure and base code.

SEO Friendly

Most frameworks render the content at runtime but componet.js works with rendered HTML so it's better for SEO!
